# 03_RELEASE_CHECKLIST

Before release, verify:

- No V2 repository was modified.
- No production output repository was modified.
- No production Notion database was used.
- DOC_ID format is correct.
- Required fields are validated.
- Draft documents are not published accidentally.
- HTML output is clean.
- Print CSS works reasonably.
- Build report is generated.
- Errors are readable.
